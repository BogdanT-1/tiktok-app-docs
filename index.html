Translator

Personal tool to dub TikTok videos into multiple languages using AI translation and voice cloning. Paste a TikTok URL, pick target languages, get back a dubbed MP4 ready to post.

Personal project. Translator is a single-developer, non-commercial tool built for personal use and small-scale experimentation. It is not a public service or a multi-tenant product.

What it does

Transcribe

Uses OpenAI Whisper to transcribe the original audio with word-level timing.

Translate

Translates the transcript into your selected European language with GPT‑4o.

Voice clone

Synthesises the dub using ElevenLabs voice cloning so the new voice matches the original speaker.

TikTok drafts

Uploads the dubbed MP4 directly to your TikTok Drafts via the Content Posting API.

How it uses TikTok

Translator connects to TikTok with the user.info.basic and video.upload scopes only. It reads your basic profile to confirm the connection, and writes dubbed MP4 files to your TikTok Drafts inbox. It never reads, modifies, or deletes existing content on your TikTok account.
